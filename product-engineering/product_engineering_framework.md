# Product Engineering Framework

## Mission
A product should have a mission driving all its efforts. This mission should target a core aspect of the direct consumer. This drives inspiration and creativity that shapes branding, consumer friendliness, etc.

## 10x Rule
A product should aim to provide 10x(value provided by current best standard). 
Whatever the current best experience consumers can get their hands on. 
This product should aim to provide 10x more value. 10 is picked because it represents 
an order of magnitude. It is the first order of magnitude which has the significance 
factor needed to be noticed by consumers. This is important because there is always a 
**switch-over cost** that a consumer will have to invest to move to a new product. 
They will have to first hear about the new product, have a good first impression, 
spend time researching and comparing, and then finally spend their money, and also install
it if necessary. 10x provides assurance for the product owner as a standard that will 
provide a more confident return for all the resource invested into creating that product.
