# The Human Model

## Internal State
The state of the human. This would be measured in the same units as paradise state. 
A delta would be able to be taken by comparing current state with ideal (paradise)state

### Human KPIs
e.g. Energy, time, capital, risk

## Huamn/Sentient Algorithm (Homeostatis)
The algorithm of life. The reason for human behaviour. 
Homeostatis, is to maintain a constant state. The state humans aim to maintain is paradise state.
(Paradise Theory)[link to paradise thoery doc]

## External State (Environment)
The state we percieve from our external environment.
## Value Primitives
A value primitive is basically the most basic fundamental unit of value that matters to the 
primary actor, the human. The human is the source of truth that defines what is considred *valuable*.
