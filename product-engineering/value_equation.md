Certainly! You can calculate the percentile rank (PR) of a number (X) in a dataset with the formula: 

\[ PR = \frac{\text{Number of values below } X}{\text{Total number of values in the dataset}} \times 100 \] 

This will give you the percentile rank of the number, indicating its rarity in the dataset.
