# Problem Document - Intent Queue

## Central problem
I need to perform so many actions daily that I lose track of them.
AI and programs can automate most of these actions, but I have to manually trigger them.
The problem is that I still have to maintain a mental queue of all the actions I need to take, which is overwhelming and inefficient.
I as a user need to define recurring tasks and a given schedule, they can also be fuzzed for randomness for certain actions that need to appear naturally timed.

## Ideal solution | Conceptual User Experience
The user defined intents should be placed in a queue, which can be pre-processed by intelligent programs to prepare the actions for execution.
The user experience should be simple and intuitive, allowing users to simply approve, edit, or reject the actions in the queue.
Every action in the queue should be executed automatically, without requiring the user to hand-hold the process.
The user only need to be involved with points of verification, since the user is the master agent.
