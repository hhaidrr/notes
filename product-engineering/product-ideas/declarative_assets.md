# Declarative Assets
The most used assets by individual consumers are their house and car.

## House
Be able to describe a house by:

* An option list (Can see a 3d model)
* etc.

Describe the state of the custom home. Have it built.
Abstracts:

* The Realtor, search and find process.

User story:

* Describe/build home
* Deal with bank and mortgage
* Move in

## Car
Be able to describe a car.

Abstracts:

* The car salesman, search and find process.

User story:

* Describe/build car
* Payment
* Delivery
