# Project Killers

## Scope Creep
Stick to your original plan. Don't add new features or change the scope of the project unless absolutely necessary.
Only iterate on the project after the MVP is complete and released.
Iterate on v1 with real user feedback.

## Perfection
Seeking perfection is a recipe for analysis paralysis, and scope creep. It halts from actually learning from real user feedback and moving to market.
