# Negative space programming
Based on NASA's 10 rules of safety critical systems
TigerBeetle modernized this into tigerstyle

introduces the concept of explicitly asserting invariants in production code. Using the assert statement in prod code.

Invariant assertions don't get rid of the need for tests, they act as a force multiplier

tests apply a discrete amount of cases upon the target scope. 

invariants live in production across all possible cases covering all of negative space. They make the code self validating
