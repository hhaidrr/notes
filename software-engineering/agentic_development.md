# Define a Software Development Life-cycle that is Driven by AI Agents

LLMs offer productive output in many use-cases. This document aims to capture different ways to integrate LLMs into
traditional software life-cycles and what pros and cons for different strategies could be.

## Full flow
1. Discuss what we want to build, having it transcribed into a raw transcript. Doing research with Deep Research as we
go
2. Have an agent take the transcript and turn it into a product specification
3. Clean up the spec, add any designs needed
4. Make a technical specification document
5. Have Devin code it.
6. QA, merge, (auto-)deploy to prod
