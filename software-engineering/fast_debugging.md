What is the lifecycle of debugging a bug?
1. experience the symptom of the bug
- first question usually is "why" is the symptom happening, we are trying to find the cause of this symptom.
... symptom chain, when do we hit the root cause, how do we know we have hit the root cause? 
... try multiple solutions, each with their own symptom and root cause iteration cycle.
... finally a solution is reached which fixes the original symptomatic behavior.
