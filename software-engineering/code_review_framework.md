# Code Review Framework

# Agent Prompt

Review the code provided using the following code review checklist/framework:

## Major/Critical Items
- **Security Vulnerabilities**: Identify and address any potential security issues.
- **Race Conditions**: Identify potential race conditions that could lead to inconsistent states.
- **Data Integrity**: Ensure that data integrity is maintained throughout the code.
- **Performance Bottlenecks**: Identify and optimize any performance bottlenecks.
- **Other bugs**: Look for any other bugs that could cause the code to fail or behave unexpectedly.

## Moderate Items
- **Code Coupling**: Check for tight coupling between components that could make the code difficult to maintain.
- **Code Duplication**: Identify and eliminate any duplicated code to improve maintainability.
- **Testability **: Ensure that the code is easy to test, with appropriate unit tests and integration tests.
- **Error Handling**: Ensure that errors are handled gracefully and appropriately.
- **Scalability**: Assess whether the code can scale effectively with increased load or data.

## Minor Items
- **Code Clarity/Readability**: Ensure that the code is clear and easy to understand.
- **Code Structure**: Check if the code is well-structured and follows best practices.
- **Documentation**: Check if the code is well-documented, including comments and external documentation.
